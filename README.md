# face & gender recognition

OpenCV를 이용하여 사진을 분석한 후,얼굴인식 및 성별 판별을 할수 있습니다. 아직 완성된 버전은 아니기에 부족한 점이 많습니다.  
사용하다 문제점이 있거나 보완할 점이 있다면 코멘트 남겨주시면 정말 감사하겠습니다. 

앞으로 꾸준히업데이트 할 예정입니다.
